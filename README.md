# Expense Tracker (Python)

A simple console-based Expense Tracker application built using Python.  
This project helps users record their daily expenses, view expense history, and calculate total spending.

##  Features
- Add new expenses
- View all recorded expenses
- Calculate total expenses
- Stores data using a text file (`expenses.txt`)
- Simple menu-driven interface

##  Technologies Used
- Python 3
- File Handling
